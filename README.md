### Project Structure

#### `io` Package

The `tictactoe.io` package handles data input and output in the command-line interface of the Tic-Tac-Toe game. It contains utilities for printing messages to the user and reading their input, keeping the presentation and data capture logic organized.
## Getting Started 

### Prerequisites

- Java Development Kit (JDK) installed on your machine. You can download it from [Oracle's official website](https://www.oracle.com/java/technologies/javase-downloads.html).

### Running the Application

1. Clone the repository or download the code files.
2. Navigate to the project directory.
3. Compile the Java code using the following command:
   ```sh
   javac Main.java
   ```
4. Run the application using the command:
   ```sh
   java Main
   ```
